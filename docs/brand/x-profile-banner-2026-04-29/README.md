## Direction

The banner is designed to match the v6 avatar direction:

- black / charcoal technical base;
- cyan and teal interface accents;
- subtle amber status points;
- terminal panels, workflow nodes, and messaging paths;
- clean negative space on the left so the X profile avatar overlay does not fight the header;
- more visual energy on the center-right and right side;
- a subtle geometric cat-bot/chat-bubble signal that echoes the avatar without becoming cartoonish.

## Prompt Brief

Create a professional modern clisbot banner for X/Twitter, wide 3:1 composition for final 1500x500 crop. Match the v6 avatar identity: black/charcoal base, crisp cyan and teal accents, developer-tool / AI agent / command-line / messaging automation feel. Leave the far left and lower-left area clean because the circular profile avatar overlays there on X. Put most visual energy in the center-right and right side. Use abstract terminal panels, subtle command prompt lines, chat-message paths, agent workflow nodes, fine grid, restrained cyan glow, small teal connectors, and a few amber micro-accent points. Include a subtle geometric cat-bot/chat-bubble silhouette echoing the avatar on the right side. Avoid readable words, slogans, fake UI labels, busy robots, stock-photo styling, purple gradients, beige/brown palette, heavy glow, and messy AI artifacts.

## Tagline Pass

v4 adds the short tagline **CLI agents in chat** on the left side, above the profile-avatar overlay area. It keeps the text minimal but visible, with a small `>_` prompt mark and a low-contrast micro line: `Slack · Telegram · coding · ops`.

## Frontier Tagline Pass

v5 changes the headline to **Frontier AI CLIs as chat agents** and the micro line to `Codex · Claude · Gemini · Slack · Telegram`.

Rationale: this is more specific than "CLI agents in chat" because it names the high-value source category, frontier AI CLIs, and the destination behavior, chat agents. It avoids the longer "ultimate agentic assistant" phrasing because X headers are read quickly and should stay concrete at small crops.
